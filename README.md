## âš™ï¸ Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/varungowda02/Build-with-AI-AI-Powered-Dashboards-with-Streamlit.git
   cd Build-with-AI-AI-Powered-Dashboards-with-Streamlit
   ```

2. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   ```

3. **Add your OpenAI API key**
   Add your openAI api key in:

   ```bash
   openai_key.txt
   ```

4. **Run the Streamlit app**

   ```bash
   streamlit run 04_06b.py
   ```
   For Deployment use ".../Chapter_4/Landon_Hotel_Dashboard.py"
   ```bash
   streamlit run Landon_Hotel_Dashboard.py.py
   ```

   ```bash
   pytest 04_05b.py
   ```
